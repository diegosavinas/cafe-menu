# Café Menu

Este es un proyecto simple que muestra un menú de café en formato HTML y CSS. Incluye una página web que enumera diferentes tipos de café y postres, junto con sus precios.

## Uso

Simplemente abre el archivo `index.html` en tu navegador web para ver el menú.

## Recursos

- [FreeCodeCamp](https://www.freecodecamp.org) - Sitio web donde se obtuvieron las imágenes utilizadas en el proyecto.
